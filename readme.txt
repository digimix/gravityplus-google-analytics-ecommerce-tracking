== Installation ==

This section describes how to install and setup the Gravity Forms + Google Analytics Ecommerce Tracking Add-On. Be sure to follow *all* of the instructions in order for the Add-On to work properly. If you're unsure on any step, there are screenshots.

Requires at least WordPress 3.9, PHP 5.3, and Gravity Forms 1.8.1. Works with WordPress Multisite.


1. Make sure you have your own copy of Gravity Forms. This plugin does not include Gravity Forms. It will work with any of the Gravity Forms licenses.

2. You'll also need a [Google Analytics](https://www.google.com/analytics/) account with *Universal Tracking* enabled.

3. Upload the plugin to your WordPress site. There are two ways to do this:

    * WordPress dashboard upload

        - Download the plugin zip file by clicking the orange download button on this page
        - In your WordPress dashboard, go to the **Plugins** menu and click the _Add New_ button
        - Click the _Upload_ link
        - Click the _Choose File_ button to upload the zip file you just downloaded

    * FTP upload

        - Download the plugin zip file by clicking the orange download button on this page
        - Unzip the file you just downloaded
        - FTP in to your site
        - Upload the `gravityplus-google-analytics-ecommerce-tracking` folder to the `/wp-content/plugins/` directory

4. Visit the **Plugins** menu in your WordPress dashboard, find `Gravity Forms + Google Analytics Ecommerce Tracking` in your plugin list, and click the _Activate_ link

5. Visit the **Forms->Settings** menu, select the new _GA Ecommerce Tracking_ tab, and follow the instructions to enable ecommerce tracking in your Google Analytics account and enter your Google Analytics tracking ID. Save your settings.

6. Create a form with your desired fields.

8. In the **Form Settings->GA Ecommerce Tracking menu**, add a new GA Ecommerce Tracking feed for your form
